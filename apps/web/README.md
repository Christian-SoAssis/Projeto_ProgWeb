# ServiçoJá — Frontend (Next.js 14)

PWA do marketplace de serviços locais. Parte do monorepo `Projeto_ProgWeb`.

## Dev

```bash
docker compose up -d
# Acesse: http://localhost:3000
```

## Stack
- Next.js 14 (App Router) + TypeScript
- shadcn/ui + Tailwind CSS
- Tema: Lavanda Pó/Noturna (neomórfico)

Consulte `openspec/` para specs de componentes e fluxos.
